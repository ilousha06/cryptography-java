###Projet de Cryptographie en Java
## Description

Ce projet a été réalisé dans le cadre d’un travail pratique de cryptographie.
Il a pour objectif de mettre en œuvre plusieurs techniques de chiffrement et de déchiffrement en Java.

Les fonctionnalités principales du projet sont :
- l’analyse fréquentielle d’un texte chiffré,
- le chiffrement et déchiffrement par substitution mono-alphabétique,
- le chiffrement et déchiffrement par XOR,
- l’attaque par texte clair connu,
- la simulation simplifiée de la machine Enigma.

Le projet est composé de deux classes principales :
- `Crypto` : regroupe les méthodes liées aux différents chiffrements.
- `Enigma` : simule le fonctionnement des rotors de la machine Enigma.

Les tests sont réalisés à l’aide de fichiers textes et les résultats sont affichés ou générés sous forme de fichiers.
